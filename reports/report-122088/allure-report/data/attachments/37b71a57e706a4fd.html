  1.         sample = new Button("Click");
  2.         sample.addClickListener(event -> Notification.show("The button was clicked", Type.TRAY_NOTIFICATION));

The Button is a basic click button. It has a caption and can also have an icon, which you can set as a Resource. Clicking a button emits a Button.ClickEvent, which can be handled with a Button.ClickListener. After clicking, the button returns to its normal state.

For styling, the Button has alternative small and link styles. The link style shows it has a hyperlink.
