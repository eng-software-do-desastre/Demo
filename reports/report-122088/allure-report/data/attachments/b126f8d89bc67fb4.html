The TextField component is a single-line editor for values. The default type for the value is String, but you can use almost any type by using a converter. The built-in converter handles most basic type conversions.

Editing a value causes a ValueChangeEvent in the edited property, which you can handle with a ValueChangeListener, as with other fields. You can also use a TextChangeListener to listen for changes interactively as they are typed.
